### PinLite — Мини-клон Pinterest для техникума

Проект создан как учебное задание для техникума. Это простой сервис для публикации и поиска фото по тегам, с авторизацией пользователей и комментариями.

## Основной функционал
- Регистрация и авторизация пользователей
- Загрузка фото с добавлением тегов и описания
- Поиск фото по тегам
- Общая лента всех опубликованных фото
- Профиль пользователя с его фото и возможностью удаления
- Комментарии к фото с возможностью ответа, редактирования и удаления
- Сохранение всех данных между перезапусками приложения

## Стек технологий
- Python 3.10+
- Flask 3.0+
- Bootstrap 5 для фронтенда
- Текстовые файлы для хранения данных (без баз данных, для простоты проверки)

## Как запустить локально

### Шаг 1: Клонируйте репозиторий
Bash

git clone https://github.com/ВАШ_НИКНЕЙМ/имя-репозитория.git
cd имя-репозитория

### Шаг 2: Создайте виртуальное окружение
В PyCharm это делается автоматически при открытии проекта. В терминале:
Bash

python -m venv venv

### Шаг 3: Активируйте окружение и установите зависимости
Bash

# Для Windows
venv\Scripts\activate

# Для Linux/macOS
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

### Шаг 4: Запустите приложение
Bash

python app.py
Откройте в браузере: http://127.0.0.1:5000

## Структура проекта
PinLite/
├── static/
│   └── uploads/       # Папка для сохраненных фото
├── templates/         # HTML-шаблоны
│   ├── login.html
│   ├── register.html
│   ├── reset_password.html
│   ├── feed.html      # Общая лента фото
│   ├── upload.html    # Страница загрузки фото
│   ├── profile.html   # Профиль пользователя
│   ├── photo_page.html# Страница отдельного фото с комментариями
│   └── edit_comment.html # Редактирование комментариев
├── app.py             # Основной код приложения
├── users.txt          # Хранилище пользователей, фото и комментариев

## Примечания для проверки
- Все данные хранятся в одном файле users.txt — нет необходимости настраивать базу данных
- После перезапуска приложения все фото и комментарии сохраняются
- Для тестирования можно использовать предопределенные аккаунты:
  - Логин: user, Пароль: user1
  - Логин: admin, Пароль: admin1
